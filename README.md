## 📌 Project Overview

An end-to-end data analytics project on 9,994 retail sales records from the Superstore dataset (2014–2017). Covers the full analytics pipeline — from raw data cleaning and SQL querying to interactive dashboard visualisation and business insight generation.

---

## 🔍 Key Findings

- 📈 **51.4% sales growth** from 2014 to 2017
- 🌍 **West region** is the highest profit-generating region
- ❌ **Tables** lost **$17,725** and **Bookcases** lost **$3,472** — identified as loss-making sub-categories
- 🏆 **Technology** is the most profitable product category
